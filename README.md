# Numerical Methods

This is a simple website that demonstrates the implementation of three numerical methods for solving nonlinear equations: the bisection method, the regula falsi method, and the Newton-Raphson method.

## Code Structure

The `index.html` file is the main HTML file for the website. It includes the necessary HTML and CSS code to create the layout of the website. The `style.css` file contains the CSS code for the website and all the respective codes for different method calculator is in the respective directory 

## Using the Website

To use the website, simply click on one of the buttons in the "Numerical Methods" section. This will take you to a page that demonstrates the implementation of the selected method.

## Conclusion

This website is a simple demonstration of the implementation of three numerical methods for solving nonlinear equations. It is intended to be used as a learning resource for junior developers.
